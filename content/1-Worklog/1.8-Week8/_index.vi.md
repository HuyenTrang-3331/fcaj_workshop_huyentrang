### Mục tiêu tuần 8:

- Hoàn thành nội dung học tập và thực hành của Module 07 về dữ liệu, phân tích dữ liệu và xử lý dữ liệu serverless trên AWS.
- Thực hành tạo S3 Bucket, Delivery Stream, sample data, Glue Crawler và kiểm tra dữ liệu đầu ra trên S3.
- Tìm hiểu quy trình phân tích dữ liệu với Amazon Athena và trực quan hóa dữ liệu bằng Amazon QuickSight.
- Thực hành với Amazon DynamoDB, bao gồm khám phá DynamoDB Console, backup, clean up và các mô hình thiết kế nâng cao.
- Tìm hiểu cách xây dựng ứng dụng serverless toàn cầu và kiến trúc event-driven với DynamoDB.
- Thực hành xây dựng database, kiểm tra dữ liệu, phân tích chi phí, tagging, usage và query trên DynamoDB.
- Thực hành sử dụng CloudShell, AWS Console, SDK và Cloud9 trong quá trình xử lý dữ liệu.
- Thực hành xử lý dữ liệu với AWS Glue DataBrew, AWS Glue, EMR, Athena, Kinesis Data Analytics, QuickSight, Lambda và Redshift.
- Rèn luyện kỹ năng xây dựng dashboard, cải thiện dashboard và tạo interactive dashboard phục vụ phân tích dữ liệu.
